# Prompt: Cari File Logo PNG Universitas Khairun

Paste prompt ini ke Claude browser/computer use.

---

## PROMPT

Bantu saya mencari file logo Universitas Khairun (PNG atau JPG) di komputer ini. Lakukan langkah berikut:

### LANGKAH 1 — Cari di folder umum

Buka File Explorer / Finder, lalu cari di lokasi berikut satu per satu:
- Desktop
- Folder Downloads
- Folder Documents / Dokumen
- Folder Pictures / Gambar

Cari file dengan nama yang mengandung kata: `unkhair`, `khairun`, `logo`, `universitas`

### LANGKAH 2 — Gunakan fitur Search / Pencarian

1. Tekan **Windows + S** (Windows) atau **Cmd + Space** (Mac)
2. Ketik: `unkhair`
3. Lihat semua hasil yang muncul — catat path lengkap file yang ditemukan
4. Ulangi dengan kata kunci: `logo khairun`
5. Ulangi dengan kata kunci: `logo universitas`

### LANGKAH 3 — Cek folder project

Buka folder project pendaftaran S2 di komputer lokal (jika ada), lalu periksa:
- Folder `public/`
- Folder `assets/`
- Folder `images/`

### LANGKAH 4 — Laporkan hasilnya

Setelah selesai mencari, laporkan:
- **Nama file** yang ditemukan (beserta ekstensi: .png, .jpg, .svg, dll)
- **Path lengkap** file tersebut (contoh: `C:\Users\nama\Downloads\logo-unkhair.png`)
- **Ukuran file** jika bisa dilihat

---

**Tujuan:** Saya perlu mengunggah file logo PNG/JPG asli Universitas Khairun ke folder `public/` di project Next.js, untuk menggantikan logo SVG rekonstruksi yang saat ini digunakan.
